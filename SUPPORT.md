# Support

Use GitHub Discussions for usage questions and design conversations when
Discussions are enabled.

Open a GitHub issue for reproducible bugs and feature requests. Include enough
information for another person to reproduce the behavior.

This community project does not currently provide guaranteed response times or
commercial support.

For vulnerabilities, follow [SECURITY.md](SECURITY.md) and do not create a
public issue.

Useful diagnostic information includes the library version, Go version, Redis
version and topology, selected failure mode, policy, framework adapter, and
relevant logs with secrets removed.
